* 新增一个页面page，你需要：
  * 创建一个目录page
  * 在page目录下创建名为page的页面（即wxml, wxss, js, json四个文件）
  * 在app.json的pages中注册页面
  * 在本表格中添加新表项
* “跳转自”不包含从下级页面返回到上级页面（事实上这个返回按钮由`navigateTo`自动产生）
